Project X

Examination project made in Unreal Engine 5, Blueprint only mini-open world rpg style game. The goal of this project was to learn as much as possible about how to make ARPG game, I was increadibly curious about how AI are made, combat and ability systems works in games.

This is my 3rd Unreal Engine project and 8 weeks of puzzling experience I managed to create a decent enemy AI system that has melee, ranged and mage classes and an interesting that demonstrates the combination of these classes.
