Undying Hope

This is another group project where we made a 3th person, open world game. In this game the player has to capture flags that are placed in different parts of the map and these capture points are heavily guarded by horde of enemies. The player has to kill enemies, sustain himself potion drops and level up to gain better stats.

This project tought me about the importance of flexiblity when it came to finding solution to a problem and thinking outside of the box.

My contributions was to shape the terrain, make level up system, make enemies spawn, design and code UI elements and the Main Menu.
